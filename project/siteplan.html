Site Name: Frontend Web Development Crash Course

Description: The site provides educational videos from other sources using API to teach about the basics of HTML, CSS, and JS.


Site purpose

The site purpose is to teach students the building blocks of frontend web development using interesting videos. It provides adequate teaching materials for students who want to learn about the basics of frontend web development.


Scenarios

What is HTML, and how do we implement it?

What is CSS, and how do we implement it?

What is JS, and how do we implement it?

How can we integrate these three components of web development to create beautiful and responsive websites?
